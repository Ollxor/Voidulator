# Smoke Test Checklist

- [ ] Loads with **1 greyscale beam**; animates.
- [ ] Beam **count** synced (slider + number).
- [ ] **Beam width** allows decimals; visual width updates.
- [ ] **Pulse**: off/sine/square; amp/freq/speed change output.
- [ ] Circle / regular N / random N / blob build correctly; bounce count updates.
- [ ] Emitters: drag/move; overlay draws/updates.
- [ ] Perf sanity: 18–24 beams, moderate width/bounces is smooth.
- [ ] Export/record works; background is black.
- [ ] No console errors.
